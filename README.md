# Personal Expense Tracker

## Overview

The Personal Expense Tracker app helps users record and manage income and spending by categorizing expenses, setting budgets, and viewing spending summaries. It includes features like receipt capture, bill reminders, and financial reports to help users control their finances and reach money goals.

## Features

- Add and categorize expenses
- Track income and spending
- Set and monitor budgets
- Capture receipts and bills
- View financial summaries and charts
- Get reminders for bills and recurring payments

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/Hamsa1016/Personal-Expense-Tracker.git
    ```
2. Navigate to the project directory:
    ```
    cd Personal-Expense-Tracker
    ```
3. Install dependencies:
    ```
    npm install
    ```
4. Start the application:
    ```
    npm start
    ```
   The app will run at `http://localhost:3000`

## Usage

- Register or log in to your account
- Add your income and expenses and assign categories
- Set budgets and track progress
- Analyze your financial habits with reports and charts
- Receive timely reminders for upcoming bills
